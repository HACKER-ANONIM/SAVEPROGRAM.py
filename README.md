# SAVEPROGRAM.py

Простая программа для резервного копирования файлов и папок на диск `D:\SERVER` (или в любую другую папку).

## Возможности
- Копирование одного файла или целой папки.
- Интерфейс на Tkinter.
- Лицензия MIT.

## Как использовать
1. Запустите SAVEPROGRAM.py
2. Нажмите «Обзор...» и выберите файл/папку.
3. Нажмите «КОПИРОВАТЬ НА D:\SERVER»

## Примечание
Папка `SERVER` создаётся автоматически на диске D:.
